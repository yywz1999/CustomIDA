# CustomIDA
### 欢迎使用CustomIDA!!
- 它可以帮助您替换IDA的图标资源文件(目前仅支持8.3).
- 这个是在学习qt开发中写的小程序.
### 注意事项:
 - 图片的大小尽量控制在128K以内.
 - 图片的尺寸最好是等比例(程序会自动进行裁剪).
 - 选择IDA.app即可, 不用选择IDA64.app
